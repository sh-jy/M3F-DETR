# M3F-DETR

Abstract：In autonomous driving perception systems, existing visual object detection algorithms still face challenges in multi-scale and complex-background scenarios, such as limited receptive coverage and insufficient context modeling. To address these issues, we propose M3F-DETR, a real-time detection Transformer architecture for complex road scenes, aiming to improve both detection accuracy and real-time performance in challenging traffic driving conditions. Specifically, we design a Long-Range Dependency Enhanced Module (LDEM) based on the Mamba sequence modeling framework, which substantially strengthens the network’s contextual perception while maintaining low latency, thereby improving real-time detection of traffic targets. In addition, the proposed Multi-Scale Semantic Fusion Structure (MSFS) dynamically guides semantic fusion across different feature hierarchies, effectively mitigating feature inconsistency among multi-scale objects. Furthermore, we develop a Frequency-Domain–Guided Decoupled Attention (FDGD) mechanism that leverages frequency-domain priors to decouple features into high- and low-frequency components and fuse them through directed interactions, achieving both fine-detail enhancement and global noise suppression with minimal overhead; this alleviates attention bias induced by multi-scale feature coupling and further improves real-time detection in complex traffic scenes. Experimental results demonstrate that M3F-DETR achieves an AP of 54.2% and a mAP of 89.28% on KITTI-ROD, while reaching an FPS of 97.1 on Cityscapes, thereby exhibiting superior detection performance and real-time capability.
